# Task 1: Weather API Integration and Visualization

## Objective
Fetch real-time weather data using a public API and visualize it using Python.

## API Used
OpenWeatherMap API

## Technologies
- Python
- Requests
- Matplotlib

## Description
This project fetches current weather data for Pune city using OpenWeatherMap API and displays temperature and humidity data in a graphical format.

## Files Included
- weather_api.py : Python script for API integration and visualization
- output.png : Graphical output generated using Matplotlib

## How to Run
1. Install required libraries:
   pip install requests matplotlib

2. Run the script:
   python weather_api.py
